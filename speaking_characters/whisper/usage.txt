Whisper basic version:
docker run -d -p 9000:9000 -e ASR_MODEL=medium -e WHISPER_LANGUAGE=fr onerahmet/openai-whisper-asr-webservice:latest


Custom Docker with Whisper for audio to text and playnote for Diarization:
  Build the docker image:
docker build --no-cache -t whisperx-lipsync .


Run the docker image:
docker run --runtime=nvidia --gpus all -it \
  -v $(pwd):/app \
  whisperx-gpu \
  whisperx episode-test-mouth-lion.mp3 \
    --model large-v2 \
    --device cuda \
    --diarize \
    --hf_token $HF_TOKEN -> get one there: https://huggingface.co/pyannote/speaker-diarization-community-1  \
    --output_dir /app/output \
    --output_format json
    --language fr
